⚙️ إعداد جدول payment_requests

هذه الصفحة تُنشئ جدول طلبات الدفع وصلاحياته في Supabase تلقائياً.

📋 ما سيتم إنشاؤه:
✅ جدول payment_requests لحفظ طلبات الدفع
✅ سياسات RLS — المستخدم يرى طلباته فقط، الأدمن يرى الكل
✅ Function + Trigger لإعطاء الرتبة تلقائياً عند الموافقة
⚠️ ملاحظة مهمة:
يجب تسجيل الدخول بحساب Admin في الموقع أولاً.
أو تشغيل هذا مباشرة من Supabase Dashboard → SQL Editor.
