## 🕹️ About Virtux

**Virtux** is a gaming community platform built for the English-speaking audience (Europe & Americas). It serves as a hub for gamers, content creators, and esports enthusiasts to discover live streams, tournaments, highlights, and collaborate with other creators.

---

## 🚀 Features

| Section | Description |
|---|---|
| 🔴 **Live** | Watch live streams, pre-lives, live news and VIP content |
| 🏆 **Tournaments** | Ongoing tournaments, global events, news and VIP brackets |
| 🎬 **Highlights** | Pro clips, tournament highlights, photos and videos |
| 👥 **Subscribers** | Youtubers, members, donors and collaborations |
| 📞 **Contact** | Discord, Email, Twitter/X, Instagram, YouTube |
